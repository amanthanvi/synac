---
id: 'aead'
term: 'AEAD (Authenticated Encryption with Associated Data)'
summary: 'AEAD provides confidentiality and integrity for a message and authenticates optional associated data that is not encrypted.'
tags: ['crypto', 'rfc']
sources:
  - kind: 'RFC'
    citation: 'RFC 5116'
    url: 'https://www.rfc-editor.org/rfc/rfc5116'
    normative: true
updatedAt: '2025-09-02T00:00:00.000Z'
---
AEAD binds associated data to the ciphertext so tampering is detectable. Common constructions include GCM and CCM.
